# Quantum-Hypercube
This repository contains software produced by [A. Kydros](https://orcid.org/0009-0009-5278-5480) and [K. Prousalis](https://orcid.org/0000-0002-6810-1719) in
conducting research on multi-dimensional quantum images and the quantum cycle shift image transformation.

The included scripts ran locally on Python version 3.11.9.
